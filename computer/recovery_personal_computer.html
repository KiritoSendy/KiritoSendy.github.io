重装系统-Personal

发表于 2020-03-02 更新于 2020-09-16 分类于 ComputerBasics
本文字数： 770 阅读时长 ≈ 1 分钟
记录一下个人电脑重装时的步骤, 以备不时之需.

账户名称设置

重装系统时, 不要用微软账户登录, 而是使用本地账户登陆 , 此时系统盘中账户文件夹的名称就是本地账户的名称, 之后再用微软账户登陆, 用户文件夹的名字就不会采用微软账户邮箱的前四个字母了.

修正盘符

右键 "此电脑" -> "管理" -> "磁盘管理", 修正磁盘盘符.

Windows 10 基础设置

进入设置, 对 Windows 10 进行基础设置, 更改桌面图标, 更改鼠标样式.

将 "回收站" 添加进 "快速访问"

向快速访问中添加 "回收站".

设置 "文件夹选项"

设置"文件夹选项". 包括:

  • 设置 Win+E 打开 "此电脑".
  • 清除文件资源管理器的历史纪录.
  • 设置 "快速访问" 不显示最近使用的文件, 文件夹.
  • 显示文件后缀名, 始终显示隐藏文件.
  • 用彩色显示加密或压缩的 NTFS 文件.
  • 在标题栏中显示完整路径.

更改 "高级系统设置"

  • 计算机名称
  • 将虚拟内存放置到固态硬盘, 空间大小 20G.
  • 在系统保护中, 开启系统盘系统保护, 空间大小最大 8G.

禁用 Windows Defender

组策略: gpedit.msc

Bandzip, HoneyView, Chrome
